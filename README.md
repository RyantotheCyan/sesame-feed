# sesame-feed

The published daily-briefing feed for the Sesame reading app.

`feed.json` is **generated — do not edit it here.** Editions are authored as
Markdown in the app repo under `content/YYYY-MM-DD.md`, compiled by
`tools/content/lexis.py`, and pushed to this repo by `make publish`.

Served at <https://ryantothecyan.github.io/sesame-feed/feed.json>.
